# ⚪🔴⚫ SPFC Connect

## 📖 Sobre o Projeto

O **SPFC Connect** é um projeto desenvolvido com o objetivo de reunir informações, curiosidades e interação entre torcedores do São Paulo Futebol Clube. A plataforma foi criada para proporcionar uma experiência intuitiva e agradável, permitindo que os usuários conheçam mais sobre a história, conquistas e ídolos do clube.

Além disso, o sistema conta com recursos de cadastro, login, quiz interativo e área de estatísticas, promovendo maior engajamento dos torcedores.

---

## 🎯 Objetivo

Desenvolver uma aplicação web completa que demonstre conhecimentos em:

* HTML
* CSS
* JavaScript
* Banco de Dados MySQL
* Node.js
* API
* Versionamento com Trello e GitHub

Ao mesmo tempo, homenageando um dos maiores clubes do futebol brasileiro: o São Paulo FC.

## 💻 Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/seu-usuario/spfc-connect.git
```

### 2️⃣ Entrar na pasta

```bash
cd spfc-connect
```

### 3️⃣ Instalar dependências

```bash
npm install
```

### 4️⃣ Configurar o banco de dados

Execute o arquivo:

```sql
script.sql
```

no MySQL Workbench.

### 5️⃣ Iniciar o servidor

```bash
npm start
```

ou

```bash
npm run dev
```
